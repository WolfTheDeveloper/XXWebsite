Welcome to the XX Wiki

About XX

XX is an open-source programming language built on Python. Its goal is to serve as a bridge between programming languages, allowing programmers to code in multiple languages within one script.

How to Contribute

To contribute, you need to be familiar with Python. Additions to XX should be universally useful. If you're not looking to contribute code but still want to help, consider opening an issue for any bugs you encounter.

XX Founder

Zander Lewis (TheMidnightWolf or WolfTheDev) started this project with assistance from ChatGPT on February 16, 2024. At 15, Zander envisioned XX as a bridge between languages, accomplishing what others could not. For more information, visit the About XX page.

How to Run XX

python main.py xxfilename.xx

For example, to run a program, use python main.py example.xx.
